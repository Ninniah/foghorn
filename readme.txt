= Foghorn =

Foghorn is a well-coded minimalist theme.  It's great for all types of blogs and websites.  There's a custom options panel for switching layouts, removing sidebars, and uploading a logo.  It's responsive design also looks excellent in all modern browsers, including iPhone and mobile devices.  View a theme demo at http://themes.wptheming.com/foghorn.

Foghorn is originally based on Twenty Eleven by the WordPress team, http://wordpress.org/.  The theme icons were adapted from WordPress UI icons designed by Ben Dunkle.

The options panel requires the Options Framework plugin: http://wordpress.org/extend/plugins/options-framework/

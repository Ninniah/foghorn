= Foghorn =

Foghorn is a well-coded minimalist theme.  It has a custom options panel for switching layouts, removing sidebars, uploading a logo, and more.  It's responsive design looks excellent in all modern browsers, including iPhone and mobile devices.  View a theme demo at http://themes.wptheming.com/foghorn.

Foghorn is originally based on Twenty Eleven by the WordPress team, http://wordpress.org/.  The theme icons were adapted from WordPress UI icons designed by Ben Dunkle.

The options panel requires the Options Framework plugin: http://wordpress.org/extend/plugins/options-framework/
